# AI 驱动学生效率工具合集
基于 Claude + DeepSeek + VS Code 开发的一站式学习效率工具套件

## 👀 项目简介
我是一名在校学生，利用 AI 辅助开发了这套面向学习生活的工具集合，解决日常学习中**任务容易忘记、文件杂乱难找、信息记录繁琐、操作效率低**的痛点，全程使用 Claude 做需求设计、DeepSeek 生成与优化代码、VS Code 完成开发调试。

---

## 🧰 包含工具列表

### 1. 桌面宠物（DeskPet）
- 功能：桌面常驻互动小宠物，可互动、学习陪伴、定时提醒、状态展示
- 特点：轻量化、无侵入、界面可爱，适合学生长时间学习使用
- 技术：Python / 桌面UI开发

### 2. 浏览器提醒与笔记插件
- 功能：在任意网页设置**稍后提醒**，并对当前页面做笔记记录
- 特点：一键保存、定时弹窗提醒、页面信息永久留存，不用再切软件记录
- 技术：浏览器扩展（Chrome/Edge）、本地存储

### 3. 智能截图工具
- 功能：快速区域截图，支持保存、标注、文字识别（OCR）
- 特点：上课截图整理笔记、资料截图一键转文字
- 技术：截图API、图像识别

### 4. AI 自动文件整理助手
- 功能：鼠标**右键菜单**触发，AI 自动识别文件内容，一键分类归档
- 特点：课件、图片、文档、安装包自动归类，告别杂乱桌面
- 技术：系统右键集成、AI 语义分类、文件操作
